## 📝 Documentation Best Practices

### When Creating New Docs:
1. Add to this index
2. Include read time estimate
3. Link to related documents
4. Update STATUS.md if relevant
5. Mark superseded docs as deprecated

### When Updating Existing Docs:
1. Update "Last Updated" date
2. Add changelog entry if major
3. Update this index if title/purpose changes
4. Verify all cross-links still work

### Deprecation Process:
1. Mark with ~~strikethrough~~ in this index
2. Add "(Deprecated)" to filename or header
3. Add pointer to replacement document
4. Keep for 1 month, then archive or delete
